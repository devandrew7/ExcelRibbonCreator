## 1. 화면 구성 및 레이아웃

웹 페이지는 직관적인 작업을 위해 다음과 같은 **4대 영역**으로 구성되어 있습니다:

1. **왼쪽: 동작 컨트롤 팔레트 (Palette Sidebar)**
   - 리본 메뉴에 추가할 수 있는 14가지 컴포넌트(일반 버튼, 토글 버튼, 체크박스, 입력 상자, 콤보 박스, 드롭다운, 하위 메뉴, 동적 메뉴, 갤러리, 버튼 그룹, 분할 버튼, 레이블 정보, 구분선, 상자 컨테이너)가 진열되어 있습니다.
2. **중앙: 리본 실시간 시뮬레이터 & 속성 에디터 (Simulator Canvas & Property Editor)**
   - 엑셀 순정 Fluent UI 디자인 테마와 동일한 레이아웃으로 리본 메뉴를 화면에 그려 보여주는 시뮬레이션 캔버스입니다.
   - 시뮬레이터 바로 아래의 **속성 에디터**를 통해 선택한 컨트롤의 ID, 표시 이름, imageMso, 크기, VBA 핸들러, 그리고 해당 컨트롤에 직접 바인딩될 **VBA 프리셋 연결 및 직접 코드 편집**을 지원합니다.
3. **중앙 하단: 자주 사용하는 VBA 매크로 라이브러리 (VBA Preset Library)**
   - 엑셀 실무에서 가장 자주 쓰이는 5가지 유틸리티 VBA 매크로 코드(시트 보호 토글, PDF 저장, 열 너비 자동 맞춤, 중복 데이터 제거, 여러 시트 데이터 병합)를 제공하며 즉석에서 복사하여 쓸 수 있습니다.
4. **오른쪽: 내보내기 및 엑셀 적용 가이드 패널 (Export & Guides)**
   - 최종 완성된 설계를 실제 엑셀 파일(`.xlsm`)로 빌드하여 다운로드할 수 있는 내보내기 단추와 4단계 적용 가이드, 그리고 차단 해제 보안 팁 카드(클릭 시 확장)가 포함되어 있습니다.

## 2. 핵심 조작법 (컨트롤 배치 및 정렬)

### 2.1 컨트롤 추가하기 (하이브리드 지원)
- **드래그 앤 드롭 추가**: 왼쪽 팔레트에서 원하는 컨트롤을 마우스로 누른 채 중앙 시뮬레이터 내부의 특정 **기능 그룹 상자** 위로 마우스를 끌어다 드롭하십시오.
  - 마우스를 올리는 순간 그룹 상자가 초록색 점선 테두리와 배경 하이라이트로 반응하여 직관적으로 배치할 수 있습니다.
- **클릭 추가**: 팔레트의 컨트롤 카드를 클릭하면 현재 시뮬레이터 상에서 활성화(선택)되어 있는 기존 탭/그룹 내부에 자동으로 추가됩니다.

### 2.2 시뮬레이터 내부 정렬 드래그 앤 드롭
- 시뮬레이터 내부에서 이미 생성된 컨트롤 카드나 수직 구분선을 마우스로 드래그하여 다른 컨트롤 위에 드롭해 위치를 즉석에서 정밀 삽입할 수 있습니다.
- 특히 **수직 구분선(`.ribbon-separator-sim`)**에 드래그 요소가 Hover되면 구분선이 **4px 크기의 선명한 엑셀 그린 색상 삽입선**으로 팽창하여 정밀 삽입 안내 역할을 제공합니다.

### 2.3 4방향 D-Pad 정밀 컨트롤 이동
시뮬레이터에서 배치된 특정 컨트롤을 클릭하면 하단에 **◀ ▲ ▼ ▶ ✖ (D-Pad)** 오버레이 바가 생성됩니다.
- **대형 컨트롤 (Large size) 및 구분선**:
   - `◀` / `▶` 버튼만 활성화됩니다. 클릭 시 컬럼 단위로 가로 순서가 한 단계씩 좌우로 바뀝니다.
- **소형 컨트롤 (Normal size)**:
   - `▲` / `▼` (위/아래): 동일한 세로 정렬 컬럼 내부에서 위아래로 한 단씩 오르내립니다.
   - `◀` / `▶` (좌/우): **컬럼 단위 건너뛰기 점프**를 수행하여, 옆 컬럼으로 소형 컨트롤 열 전체를 안전하게 건너 이동시킵니다.
- **✖ (삭제)**: 해당 컨트롤을 즉각 영구 삭제합니다.

## 3. 신규 프리미엄 컨트롤 설명

새롭게 지원하는 맞춤형 제어 컨트롤의 기능과 XML 빌드 규격은 다음과 같습니다:

1. **동적 메뉴 (RibbonDynamicMenu) ⚡**:
   - **설명**: 고정된 하위 단추 대신, 엑셀 구동 런타임에 VBA 콜백 함수를 통하여 동적으로 생성한 XML 내용을 기반으로 메뉴 목록을 조립하여 출력해 주는 고도의 컨트롤입니다.
   - **속성**: `getContent` 속성에 바인딩할 VBA 콜백 함수명(예: `GetDynamicContent`)을 입력합니다.
   - **VBA 콜백**: 자동으로 다음 시그니처의 VBA 코드가 완성되어 모듈에 복사할 수 있게 해줍니다.
     ```vba
     Sub GetDynamicContent(control As IRibbonControl, ByRef content)
         ' ByRef content 매개변수에 주입할 메뉴 XML을 할당하여 전달합니다.
         content = "<menu xmlns=""http://schemas.microsoft.com/office/2009/07/customui"">" & _
                   "  <button id=""dyn1"" label=""동적 실행 1"" imageMso=""HappyFace"" onAction=""Dyn_Click"" />" & _
                   "</menu>"
     End Sub
     ```
2. **버튼 그룹 (RibbonButtonGroup) ⛓️**:
   - **설명**: 소형 크기(Normal) 버튼들을 가로 한 줄로 촘촘하고 얇게 묶어 정렬하여, 넓은 공간 낭비 없이 리본바를 콤팩트하게 꾸밀 때 유용한 레이아웃 래퍼 요소입니다.
3. **분할 버튼 (RibbonSplitButton) 🥞**:
   - **설명**: 왼쪽(또는 상단)의 메인 동작 단추 영역과 오른쪽의 작은 화살표 드롭다운 메뉴 영역이 결합된 형태입니다. 사용자가 메인 아이콘을 누르면 기본 VBA 동작이 바로 실행되고, 화살표를 누르면 추가 서브 단추 목록이 펼쳐집니다.

## 4. 갤러리 색상 팔레트 프리셋 활용법

갤러리(`gallery`) 컨트롤을 선택했을 때, 매번 많은 하위 아이템을 수작업으로 입력하지 않고 **실무용 색상 팔레트 그리드**를 1초 만에 구성하는 방법입니다.

1. 시뮬레이터에서 갤러리(`gallery`) 컨트롤을 추가하고 클릭합니다.
2. 속성 에디터 내의 **[색상 팔레트 프리셋 빠른 적용]** 선택 상자를 클릭합니다.
3. 다음 4가지 핵심 프리셋 중 원하는 구성을 선택합니다:
   - **기본 10색 (일정 색상)**: Excel 기본 아웃룩 일정 약속 분류 컬러(AppointmentColor1~10).
   - **표준 10색 (무지개색)**: 선명한 10가지 표준 원색 코드 매핑.
   - **오피스 테마 20색**: 10색 테마 메인 컬러와 10색 연한 틴트 색상 조합(10열 배치).
   - **화려한 40색 그리드**: 40가지 확장 색상 모음(8열 배치).
4. 적용 즉시 **[아이템 목록 편집기]**에 해당 색상의 명칭 및 16진수 색상 코드(`#HEX`)가 자동 주입됩니다.
5. **실시간 시각 렌더링**: 시뮬레이터 상의 갤러리 드롭다운을 열어보면, 해당 Hex 컬러 규격에 맞춰 **그리드 타일 안에 실제 배경 색상이 실시간 렌더링**되어 나타납니다. 흰색/밝은 색의 경우 테두리가 자동으로 형성되어 시인성을 높여 줍니다.

## 5. 토글 버튼의 라디오 그룹화 사용법

동일 그룹 내에서 오직 단 하나만 체크된 상태(pressed)를 유지하고 나머지는 체크 해제되는 **라디오 버튼(단일선택) 기능**을 구현하는 방법입니다.
1. 시뮬레이터에서 토글 버튼(`togglebutton`)을 추가합니다.
2. 속성 편집창의 **[라디오 그룹명 (toggleGroup)]** 입력 란에 원하는 그룹 식별 명칭(예: `themeGroup`)을 입력합니다.
3. 동일하게 한 그룹으로 묶고 싶은 다른 토글 버튼들도 동일한 그룹명(`themeGroup`)을 속성창에 기재합니다.
4. 이제 시뮬레이터 본문이나 속성창에서 어떤 토글 버튼을 활성화(`checked`)하면, **동일 그룹에 속한 다른 토글 버튼이 물리적으로 쑥 풀리며(checked = false) 단 하나만 쑥 눌린 입체 섀도우 효과(`.pressed` 스타일)를 유지**하게 됩니다.

## 6. Excel COM 차단 우회 및 아이콘 미리보기 활성화 방법 (중요)

> [!IMPORTANT]
> 윈도우 환경 보안 및 오피스 백그라운드 COM 인스턴스 제한(HRESULT `0x8000FFFF` / `E_UNEXPECTED`)으로 인해 파워쉘만으로 다이렉트 아이콘을 추출하면 0개 성공으로 끝나는 장애가 발생할 수 있습니다. 
> 이를 해결하려면 다음의 **VBA 30초 우회 방법**을 순서대로 진행해 주십시오.

### 6.1 단계: VBA 매크로를 이용한 BMP 초고속 추출
1. Microsoft Excel 프로그램을 직접 실행합니다.
2. 빈 통합 문서를 하나 개설한 후, 파일 형식을 **[Excel 매크로 사용 통합 문서 (*.xlsm)]**로 선택하여 **프로젝트 루트 폴더(`D:\Antigravity\ExcelRibbonCreator`)에 저장**합니다. (반드시 이 폴더에 저장해야 폴더 내 `image_mso_list.js`를 감지할 수 있습니다.)
3. <kbd>Alt + F11</kbd> 단축키를 눌러 **Microsoft Visual Basic for Applications** 에디터 창을 엽니다.
4. 상단 메뉴의 `[삽입] → [모듈]`을 클릭하여 빈 모듈(`Module1`)을 추가합니다.
5. 프로젝트 폴더 내의 **[extract_helper.txt](file:///D:/Antigravity/ExcelRibbonCreator/extract_helper.txt)** 파일 내용을 모두 메모장으로 열어 복사한 뒤, 엑셀 모듈 코드 창에 붙여넣습니다.
6. 코드 창의 `ExtractIconsToBmp` 서브루틴 안에 마우스 커서를 올린 상태에서 **<kbd>F5</kbd> 키**를 누르거나 상단의 초록색 재생 버튼(▶)을 클릭합니다.
7. 진행 팝업 창이 뜨면 `예(Yes)`를 클릭합니다. Excel 화면이 깜빡이지 않도록 처리된 상태에서 약 30~50초 만에 3,245개 전체 아이콘이 `imageMSO/` 폴더에 `.bmp` 이미지 파일로 빠르게 저장됩니다.
8. 추출이 모두 끝나면 "Extraction Complete!" 메시지 창이 표시됩니다. Excel 프로그램을 닫아 주십시오.

### 6.2 단계: 파워쉘을 통한 transparent PNG 일괄 변환 및 정리
1. 로컬 터미널(PowerShell) 창을 실행하고 프로젝트 폴더로 이동합니다.
2. 아래의 명령어를 입력하고 실행합니다.
   ```powershell
   powershell -ExecutionPolicy Bypass -File .\extract_icons.ps1
   ```
3. 스크립트가 실행되는 즉시 **"BMP to PNG Converter Mode Active"**가 동작합니다.
4. 방금 전 매크로로 생성한 모든 `.bmp` 파일을 탐색하여, .NET의 GDI+ 엔진을 활용해 투명 배경이 지원되는 고해상도 `.png` 파일로 일괄 고속 변환하고 임시 BMP 파일들을 말끔히 삭제 청소합니다.
5. 변환이 완료되면 웹 브라우저 화면의 아이콘 피커 및 리본바에서 모든 실시간 아이콘 그래픽을 깨짐 없이 완벽하게 사용할 수 있습니다!

## 7. 최종 파일 패키징 및 엑셀 적용 방법

### 7.1 파일 내보내기 (Export)
우측 상단의 **[리본 메뉴 주입 빈 .xlsm 다운로드]** 단추를 누릅니다.
- 브라우저에 내장된 패키징 모듈(JSZip)이 작동하여, 사용자가 제작한 XML 및 리본 설정이 주입된 완벽한 규격의 파일이 즉시 다운로드됩니다.

### 7.2 엑셀 프로그램에 리본 메뉴 활성화하기
1. **내보낸 파일 열기**: 다운로드된 `.xlsm` 파일을 Excel 프로그램으로 실행합니다.
2. **매크로 사용 허용**: 파일 오픈 시 상단 경고 창의 **[콘텐츠 사용]** 또는 **[매크로 활성화]** 단추를 반드시 클릭합니다.
3. **VBA 코드 주입**:
   - 단축키 `<Alt + F11>`을 눌러 VBA 프로젝트 창을 실행합니다.
   - 상단 메뉴의 **[삽입] -> [모듈]**을 눌러 빈 새 모듈(`Module1`)을 추가합니다.
   - 웹사이트 최하단 **[VBA callbacks]** 탭에 가공 생성되어 있는 VBA 소스 코드를 전체 복사(`Copy`)하여 추가한 엑셀 모듈창에 그대로 붙여넣기(`Ctrl + V`)하고 저장합니다.
4. **리본 테스트**: 이제 생성된 맞춤형 탭의 버튼, 체크박스, 토글 등을 조작할 때마다 추가해 둔 VBA 함수들이 정상 호출되어 실시간 기능이 실행됩니다!

## 8. 자주 묻는 질문 (FAQ)

#### Q. 다운로드한 `.xlsm` 파일을 실행했는데 리본 탭에 맞춤형 탭이 뜨지 않습니다.
A. 엑셀의 매크로 보안 설정에 의해 외부에서 다운로드된 매크로 파일의 실행이 비활성화되었을 수 있습니다.
- 다운로드한 파일의 우클릭 -> **[속성]**을 클릭합니다.
- 일반 탭 가장 하단의 보안 영역에서 **[차단 해제(Unblock)]** 체크박스에 체크한 후 확인을 누르고 파일을 다시 열어 보십시오.

#### Q. 탭 메뉴 구성을 고치면 엑셀 파일을 다시 다운로드 받아야 하나요?
A. 예, 리본 구조 설정은 엑셀 파일 내부의 XML 데이터에 박혀 저장되므로, 탭 구조나 버튼 속성을 변경하셨다면 웹 빌더에서 **[.xlsm 다운로드]**를 다시 눌러 최신 본을 확보하시는 것이 가장 직관적이고 확실합니다.
