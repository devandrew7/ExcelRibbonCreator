# Excel Custom Ribbon Creator 사용 설명서 (User Guide)

## 1. 화면 구성 및 레이아웃

웹 페이지는 직관적인 작업을 위해 다음과 같은 **4대 영역**으로 구성되어 있습니다:

1. **왼쪽: 동작 컨트롤 팔레트 (Palette Sidebar)**
   - 리본 메뉴에 추가할 수 있는 10가지 컴포넌트(일반 버튼, 토글 버튼, 체크박스, 입력 상자, 콤보 박스, 드롭다운, 하위 메뉴, 분할 버튼, 레이블 정보, 구분선)가 드래그 가능한 카드 형태로 진열되어 있습니다.
2. **중앙: 리본 실시간 시뮬레이터 (Simulator Canvas)**
   - 엑셀 순정 Fluent UI 디자인 테마와 동일한 폰트(11px) 및 레이아웃으로 리본 메뉴를 화면에 직접 그려 보여주는 시뮬레이션 캔버스입니다.
3. **오른쪽: 속성 편집창 및 가이드 패널 (Property Editor & Guide)**
   - 시뮬레이터에서 탭, 그룹, 개별 컨트롤을 마우스로 클릭하면 해당 요소의 상세 속성(ID, 표시 이름, imageMso, 크기, VBA 핸들러, 라디오 그룹명 등)을 편집할 수 있는 입력 창이 노출됩니다.
4. **하단: 소스 코드 및 패키지 내보내기 패널 (Output & Packager)**
   - 시뮬레이터 조작과 동시에 실시간 컴파일된 `CustomUI XML` 코드 및 `VBA Macro Callbacks` 코드가 실시간 갱신 출력됩니다.
   - 최종 완성된 설계를 실제 엑셀 파일(`.xlsm`, `.xlam`)로 패키징하여 다운로드할 수 있는 내보내기 단추가 결합되어 있습니다.

## 2. 핵심 조작법 (컨트롤 배치 및 정렬)

### 2.1 컨트롤 추가하기 (하이브리드 지원)
- **드래그 앤 드롭 추가**: 왼쪽 팔레트에서 원하는 컨트롤을 마우스로 누른 채 중앙 시뮬레이터 내부의 특정 **기능 그룹 상자** 위로 마우스를 끌어다 드롭하십시오.
  - 마우스를 올리는 순간 그룹 상자가 초록색 점선 테두리와 부드러운 배경 하이라이트(`.drag-over` 상태)로 즉각 반응하여 안전한 드롭을 돕습니다.
- **클릭 추가**: 팔레트의 컨트롤 카드를 가볍게 툭 클릭하면 현재 시뮬레이터 상에서 활성화(선택)되어 있는 기존 탭/그룹 내부에 자동으로 추가됩니다.

### 2.2 시뮬레이터 내부 정렬 드래그 앤 드롭
- 시뮬레이터 내부에서 이미 생성된 컨트롤 카드나 수직 구분선을 마우스로 드래그하여 다른 컨트롤 위에 드롭해 위치를 즉석에서 맞바꾸거나 정밀 삽입할 수 있습니다.
- 드래그 도중 대상 컨트롤 위로 지나가면 점선 테두리로 하이라이트가 표시됩니다.
- 특히 **수직 구분선(`.ribbon-separator-sim`)**에 드래그 요소가 Hover되면 구분선이 **4px 크기의 선명한 엑셀 그린 색상 삽입선**으로 팽창하여 정밀 삽입 안내 역할을 제공합니다.

### 2.3 4방향 D-Pad 정밀 컨트롤 이동
시뮬레이터에서 배치된 특정 컨트롤을 클릭하면 하단에 **◀ ▲ ▼ ▶ ✖ (D-Pad)** 오버레이 바가 생성됩니다.
- **대형 컨트롤 (Large size) 및 구분선**:
  - `◀` / `▶` 버튼만 활성화됩니다. 클릭 시 컬럼 단위로 가로 순서가 한 단계씩 좌우로 바뀝니다.
- **소형 컨트롤 (Normal size)**:
  - 세로 3단 적층 스택으로 자동 배치되는 소형 컨트롤들의 가로/세로 이동을 지원합니다.
  - `▲` / `▼` (위/아래): 동일한 세로 정렬 컬럼 내부에서 위아래로 한 단씩 오르내립니다.
  - `◀` / `▶` (좌/우): **컬럼 단위 건너뛰기 점프**를 수행하여, 옆 컬럼으로 소형 컨트롤 열 전체를 안전하게 건너 이동시킵니다.
- **✖ (삭제)**: 해당 컨트롤을 즉각 영구 삭제합니다.

## 3. M365 고해상도 로컬 아이콘 및 이모지 적용법

### 3.1 M365 로컬 아이콘 스캐너 연동
본 프로그램은 사용자의 로컬 `icons/2024-microsoft-365-content-icons/` 디렉토리에 위치한 모든 고해상도 SVG 아이콘을 웹 화면 탭으로 자동 바인딩합니다.
1. `icons/` 폴더 내에 원하는 하위 폴더들을 다중 계층 구조로 배치합니다. (예: `icons/2024-microsoft-365-content-icons/Microsoft Blue/48x48 Light Blue Icon/*.svg`)
2. 프로젝트 루트 폴더에 위치한 **`generate_icons_map.ps1` 파워셸 파일**을 마우스 우클릭한 후 **[PowerShell로 실행]**을 선택합니다.
3. 스크립트가 실행되면 폴더를 재귀 순회하여 `icons_map.js` 데이터 브릿지 파일을 자동으로 생성 및 갱신해 줍니다.
4. 이제 웹브라우저 창을 새로고침하고 속성창의 **[아이콘 선택]** 팝업 모달을 띄우면, **스캔한 폴더 구조 그대로 Level 1 서브탭과 Level 2 상세 서브탭이 동적 조립**되어 화면에 나타납니다. 원하는 아이콘 카드를 누르면 즉각 매핑됩니다.

### 3.2 윈도우 스타일 Emoji 기호 적용
- 아이콘 선택 모달 상단의 **[Emoji 이모지 기호]** 탭을 선택합니다.
- 이모지가 윈도우 단축키 `<Win + .>` 테마 분류에 준하는 **5대 그룹(웃는얼굴, 동물자연, 피플활동, 사물, 기호)**으로 깔끔하게 나누어져 제공됩니다.
- 원하는 분류 단추를 클릭해 해당 기호를 그리드 상에서 골라 리본에 삽입할 수 있습니다.

## 4. 토글 버튼의 라디오 그룹화 사용법

동일 그룹 내에서 오직 단 하나만 체크된 상태(pressed)를 유지하고 나머지는 체크 해제되는 **라디오 버튼(단일선택) 기능**을 구현하는 방법입니다.
1. 시뮬레이터에서 토글 버튼(`togglebutton`)을 추가합니다.
2. 속성 편집창의 **[라디오 그룹명 (toggleGroup)]** 입력 란에 원하는 그룹 식별 명칭(예: `themeGroup`)을 입력합니다.
3. 동일하게 한 그룹으로 묶고 싶은 다른 토글 버튼들도 동일한 그룹명(`themeGroup`)을 속성창에 기재합니다.
4. 이제 시뮬레이터 본문이나 속성창에서 어떤 토글 버튼을 활성화(`checked`)하면, **기존에 활성화되어 있던 다른 토글 버튼이 물리적으로 쑥 풀리며(checked = false) 단 하나만 쑥 눌린 입체 섀도우 효과(`.pressed` 스타일)를 유지**하게 됩니다.

## 5. 최종 파일 패키징 및 엑셀 리본 메뉴 활성화 방법

### 5.1 파일 내보내기 (Export)
하단 소스코드 탭 우측에 위치한 **[엑셀 매크로 문서(.xlsm) 내보내기]** 또는 **[엑셀 추가기능 파일(.xlam) 내보내기]** 단추를 누릅니다.
- 브라우저에 내장된 패키징 모듈(JSZip)이 작동하여, 사용자가 제작한 XML 및 리본 코드가 포함된 `customUI/customUI.xml` 구조와 매크로 설정이 주입된 완벽한 규격 파일이 즉시 다운로드됩니다.

### 5.2 엑셀 프로그램에 리본 메뉴 활성화하기
다운로드한 통합 문서를 엑셀에서 열어 리본 메뉴가 출력되는지 검사하고 VBA 이벤트 매크로를 연동하는 순서입니다.

> [!IMPORTANT]
> 엑셀에서 VBA 매크로를 작성해 두었던 컨트롤의 매크로 버튼을 눌렀을 때 오류 없이 완벽히 동작하도록 하기 위해 다음 절차를 진행하십시오.

1. **내보낸 파일 열기**: 다운로드된 `.xlsm` 파일을 Excel 프로그램으로 실행합니다.
2. **매크로 사용 허용**: 파일 오픈 시 상단 경고 창의 **[콘텐츠 사용]** 또는 **[매크로 활성화]** 단추를 반드시 클릭합니다.
3. **VBA 코드 주입**:
   - 단축키 `<Alt + F11>`을 눌러 VBA 프로젝트 창(Microsoft Visual Basic for Applications)을 실행합니다.
   - 상단 메뉴의 **[삽입] -> [모듈]**을 눌러 빈 새 모듈(`Module1`)을 추가합니다.
   - 웹사이트 최하단 **[VBA callbacks]** 탭에 가공 생성되어 있는 VBA 소스 코드를 전체 복사(`Copy`)하여 추가한 엑셀 모듈창에 그대로 붙여넣기(`Ctrl + V`)하고 저장합니다.
4. **리본 테스트**: 이제 생성된 맞춤형 탭의 버튼, 체크박스, 토글, 콤보박스 등을 조작할 때마다 추가해 둔 VBA 함수들이 정상 호출되어 실시간 로그 창이 뜨거나 기능이 실행됩니다!

## 6. 자주 묻는 질문 (FAQ)

#### Q. 로컬 SVG 폴더에 새 아이콘을 넣었는데 웹 사이트 팝업창에 안 나타납니다.
A. 웹 브라우저는 로컬 드라이브의 실제 파일 변화를 직접 감지할 수 없습니다. 아이콘 파일 추가/이름 변경 후에는 반드시 프로젝트 폴더 내 **`generate_icons_map.ps1` 파워셸 파일**을 마우스 우클릭하여 **[PowerShell로 실행]** 단추를 다시 한번 수행하여 브라우저용 데이터 브릿지 맵(`icons_map.js`)을 갱신해 주셔야 합니다.

#### Q. 다운로드한 `.xlsm` 파일을 실행했는데 리본 탭에 맞춤형 탭이 뜨지 않습니다.
A. 엑셀의 매크로 보안 설정에 의해 외부에서 다운로드된 차단 해제되지 않은 매크로 파일의 실행이 비활성화되었을 수 있습니다.
- 다운로드한 파일의 우클릭 -> **[속성]**을 클릭합니다.
- 일반 탭 하단의 보안 영역에서 **[차단 해제(Unblock)]**에 체크한 후 확인을 누르고 파일을 다시 열어 보십시오.
