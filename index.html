📚 엑셀 리본 메뉴 적용 가이드 (초보자용)
1단계: 웹에서 내보내기 & 복사 ▼
좌측에서 탭과 버튼들을 조합한 뒤, [리본 메뉴 주입 빈 .xlsm 다운로드]를 클릭해 가상 엑셀 파일을 다운로드합니다. 이와 함께 오른쪽 [VBA Callback] 탭의 코드를 복사(Copy)하여 메모장에 임시 보관합니다.
2단계: 매크로 편집기 코드 삽입 ▼
다운로드받은 엑셀 파일을 열고 단축키 Alt + F11을 눌러 매크로 편집기(VBE)를 켭니다. 상단 메뉴에서 [삽입] → [모듈]을 선택하고, 하얀 백지에 복사해둔 VBA 콜백 코드를 그대로 붙여넣습니다(Ctrl + V).
3단계: AI 비서 기능 결합 ▼
Sub 버튼명_Click()과 End Sub 사이의 빈 공간에 버튼이 작동할 실제 동작 명령어를 추가합니다. "특정 폴더를 여는 엑셀 VBA 한 줄을 짜줘" 혹은 "피벗 테이블을 자동 갱신하는 엑셀 VBA를 짜줘"라고 ChatGPT 등 AI에게 질문한 뒤, core 기능 코드를 모듈 모공 사이에 집어넣으면 완료됩니다.
4단계: 무한 영구 메뉴 등록 ▼
모든 엑셀 문서에서 이 메뉴를 평생 보려면, 작성한 통합 문서를 [다른 이름으로 저장] → 파일 형식을 [Excel 추가 기능 (*.xlam)]으로 변경해 저장합니다. 그 후 엑셀 [옵션] → [추가 기능] → 하단 관리 [이동]에서 등록하고 확인하면 평생 컴퓨터에 유지됩니다!
⚠️ 다운로드한 파일이 안 열리나요? 윈도우 보안 정책에 따라, 인터넷에서 받은 매크로 파일(.xlsm, .xlam)은 차단될 수 있습니다. 다운로드한 파일을 실행하기 전 [파일 마우스 우클릭] → [속성] → 일반 탭 가장 하단에 있는 [차단 해제 (Unblock)] 체크박스에 체크한 후 확인을 클릭해 주시면 정상 실행됩니다!
📁 아이콘 폴더 자동 동기화 가이드: M365 아이콘 폴더 내 구조를 임의로 변경(추가/삭제)하신 경우, 폴더의 generate_icons_map.ps1 스크립트를 마우스 우클릭하여 [PowerShell로 실행]해 주시면 즉시 웹사이트 탭 구조가 자동 리빌드됩니다!
