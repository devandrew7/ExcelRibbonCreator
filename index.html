📚 엑셀 리본 메뉴 적용 가이드
1단계: 웹에서 내보내기 & 복사
좌측에서 탭과 버튼들을 구성한 뒤, [리본 메뉴 주입 빈 .xlsm 다운로드]를 클릭해 엑셀 파일을 다운로드하고 하단 [VBA Callback] 탭의 코드를 복사합니다.
2단계: 매크로 편집기 코드 삽입
다운로드한 엑셀 파일을 열고 단축키 Alt + F11을 눌러 매크로 편집기를 실행합니다. 상단 메뉴에서 [삽입] → [모듈]을 추가한 뒤 복사한 코드를 그대로 붙여넣습니다 (Ctrl + V).
3단계: 실제 동작 명령어 삽입
생성된 Sub 함수명_Click() 모듈 사이에 시트 정렬, 파일 취합 등 실제로 작동할 매크로 실행 코드를 채워넣습니다.
4단계: 무한 영구 메뉴 등록 (추가 기능)
모든 엑셀 파일에서 이 리본 메뉴를 사용하려면 파일을 [Excel 추가 기능 (*.xlam)] 형식으로 저장하고, 엑셀 [옵션] → [추가 기능] → [이동]에서 이를 등록하여 사용합니다.
⚠️ 다운로드한 파일이 안 열리나요? ▼
윈도우 보안 정책에 따라, 인터넷에서 받은 매크로 파일(.xlsm)은 실행이 차단될 수 있습니다.
실행 전 [다운로드 파일 우클릭] → [속성] → 일반 탭 가장 하단에 있는 [차단 해제 (Unblock)] 체크박스에 체크한 후 확인을 클릭해 주시면 정상 실행됩니다!
